Run Pulse AI locally

The live demo runs as a real Streamlit app — instant startup, no browser runtime download needed.

1
Install dependencies
Run once in your terminal from the project folder:
pip install streamlit requests markdown fpdf2
2
Start the Streamlit app
streamlit run app/pulse_app.py
Streamlit will open at http://localhost:8501 by default.
3
Click Connect below
The app loads instantly here inside an embedded frame — no waiting.

The URL stays in your browser only — nothing is sent anywhere.
